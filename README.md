<h2>🔍 Project Overview</h2>
<p>This **fully interactive dashboard** provides deep insights into:</p>
<ul>
    <li>✅ <b>Sales Performance</b> – Category trends, product revenue breakdowns.</li>
    <li>✅ <b>Customer Demographics</b> – Buying behavior based on age, gender, region.</li>
    <li>✅ <b>Store Performance</b> – Store efficiency, size impact on revenue.</li>
    <li>✅ <b>Order & Delivery Analysis</b> – On-time vs. delayed orders.</li>
</ul>

<hr>

<h2>📂 Dataset & Data Modeling</h2>
<ul>
    <li>🔹 **Multi-Table Dataset:** Includes <b>Sales, Customers, Products, Exchange Rates, Stores</b>.</li>
    <li>🔹 **One-to-Many & Many-to-Many Relationships** handled using **Exchange Bridge Table**.</li>
    <li>🔹 **Key Transformations:**
        <ul>
            <li>✅ Created <b>Customer Age Groups</b> for segmentation.</li>
            <li>✅ Calculated <b>Profit, CLV, Store Age</b> for deeper insights.</li>
            <li>✅ Applied <b>Currency Conversion</b> using Exchange Rates.</li>
        </ul>
    </li>
</ul>

<h2>📈 Key Insights & Business Value</h2>
<ul>
